# ecommerce_vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

The Vite dev server will start with Hot Module Replacement (HMR) enabled, which means changes to your code will be reflected instantly in the browser without a full page reload.

### Preview production build
```
npm run preview
```

### Compiles and minifies for production
```
npm run build
```

### Environment Variables
Create a `.env` file in the `ecommerce_vue` directory based on `.env.example`:
```
VITE_API_URL=http://localhost:8000
VITE_STRIPE_PUBLISHABLE_KEY=your_stripe_key_here
```

### Customize configuration
See [Vite Configuration Reference](https://vitejs.dev/config/).
